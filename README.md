# DONIFY — GLOBAL IMPACT HUB

**Donify** is a prestigious, high-performance platform designed to bridge the gap between visionary "Porters" (campaigners) and institutional "Donors." Built with the **Luxury Raja** design aesthetic, it prioritizes architectural clarity, professional high-contrast interfaces, and institutional authority.

---

## 🏛 Platform Vision

Donify is not just a crowdfunding tool; it is a **Global Mission Hall**. We empower organizations and individuals to deploy high-impact initiatives through a disciplined, review-driven protocol.

### Key Pillars:

- **Heritage Design**: A "Luxury Heritage" aesthetic featuring Off-Black gradients, Emerald accents, and Beige dual-tone surfaces.
- **Architectural Clarity**: Disciplined `rounded-xl` geometry and bold structural borders.
- **Secured Protocols**: JWT-powered authentication ensuring secure transactions and identity verification.

---

## 🛠 Strategic Features

### 1. Global Mission Hall (Discovery Portal)

A cinematic discovery interface where donors can analyze tracked missions.

- **Star Protocol**: A personalized watchlist system for monitoring high-impact initiatives.
- **Telemetry Hub**: Real-time stats on tracked vs. authorized missions.
- **Intelligent Sorting**: Organize initiatives by active progress or goal magnitude.

### 2. Campaign Intelligence Console (Details Page)

A deep-dive analysis portal for individual campaigns, providing monumental data visualization and direct pledge support protocols.

### 3. Porter Command Dashboard

A specialized production terminal for campaign creators to initialize missions, track funding in real-time, and manage their impact portfolio.

### 4. Institutional Registration

A sophisticated, cinematic registration flow for organizations, featuring high-intensity dark themes and manual admin authorization protocols.

## 🏗 Technology Stack

- **Core Engine**: [Laravel 11.x](https://laravel.com)
- **Styling Architecture**: [Tailwind CSS](https://tailwindcss.com) (Strictly utility-first, "No-CSS" policy)
- **Communication Protocol**: Native AJAX/Fetch with bespoke `ApiClient` wrapper
- **Identity & Security**: [PHPOpenSourceSaver JWT-Auth](https://github.com/PHP-OpenSource-Saver/jwt-auth)
- **Typography**: Quicksand (Geometric Sans Serif)

## 🚀 Deployment & Initialization

### Environment Requirements

- PHP 8.2+
- Node.js & NPM
- Composer

### Installation Protocol

1. **Clone the Repository**:

    ```bash
    git clone https://github.com/your-repo/donify.git
    cd donify
    ```

2. **Deploy Dependencies**:

    ```bash
    composer install
    npm install && npm run build
    ```

3. **Initialize Core Systems**:

    ```bash
    cp .env.example .env
    php artisan key:generate
    php artisan jwt:secret
    ```

4. **Database Migration & Seeding**:

    ```bash
    php artisan migrate --seed
    ```

5. **Initialize Storage Bridge**:
    ```bash
    php artisan storage:link
    ```

## 📜 Aesthetic Principles (Graphic Charter)

Contributors must strictly adhere to the **Luxury Heritage** design system:

- **Primary Contrast**: White (`#FFFFFF`) / Off-Black (`#1A1A1A`)
- **Accent Materials**: Emerald Green (`#064e3b`), Metallic Gold (`#DAA520`)
- **Surface Strategy**: Dual-tone Beige (`#fbf8f6`) for data-heavy zones.
- **Radius Policy**: Fixed `rounded-xl` for all structural cards and inputs.
